Gamer Accounts System
Description

This project is a Python program that manages gamer accounts.
It reads account data from a file and allows the user to view, add, delete, and update records.

Features
View all account data
Add a new account
Delete an existing account
Update account status
Save changes to file
Data Format

Each record in the file contains:
ID, Paid Membership, Days Since Password Reset, Status

Example:
CAS10456,No,198,Active

How to Run
Open the project in PyCharm
Run the main Python file
Follow the menu options displayed
